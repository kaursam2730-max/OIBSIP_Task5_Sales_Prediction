# Sales Prediction Using Python

## Objective
The objective of this project is to predict product sales based on advertising expenditures using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Advertising Dataset

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Sales Prediction
8. Model Evaluation

## Machine Learning Algorithm
- Linear Regression

## Results
The model successfully predicts sales based on advertising budgets and demonstrates the relationship between marketing expenditure and product sales.
